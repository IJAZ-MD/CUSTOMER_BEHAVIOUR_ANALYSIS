# CUSTOMER_BEHAVIOUR_ANALYSIS
## Overview
A Power BI dashboard analyzing customer purchasing behavior, subscription trends, and review ratings using Kaggle data (3,901 rows).

## Objectives
- Identify top revenue categories
- Compare subscriber vs non-subscriber behavior
- Evaluate customer satisfaction

## Tools
- Excel & Power Query (data cleaning)
- Power BI (dashboarding)
- DAX Measures (analysis)

## Key Features
- KPIs: 3.9K customers, $59.76 avg purchase, 3.75 avg rating
- Visuals: Subscription pie chart, revenue bar charts, sales area chart
- Filters: Subscription, Gender, Category
